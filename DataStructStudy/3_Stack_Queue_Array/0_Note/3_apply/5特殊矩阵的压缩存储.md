3. 思考

   1. 数组大小应为多少？

      > $1+2+3+\cdots+n=\frac{n(n+1)}{2}$
      >
      > 因此最后一个数组元素的下标是$\frac{n(n+1)}{2}-1$

   2. 站在程序员的角度，对称矩阵压缩存储后怎样才能方便使用？

      > 可以实现一个“映射”函数
      >
      > 矩阵下标——>一维数组下标
